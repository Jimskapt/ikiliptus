Ikiliptus

An offline and mobile-friendly, time tracker and analyzer, without server needed.

Your data is your data !

Free and open-source

We cannot assure you that we don't see your data if we cannot prove it to you !

So, you can get, check and edit the recipe of this software !

And use it as you want thanks to the MIT licence !

No need of network

Offline-first credo

The application is designed to work offline, so you can track you time even offline.

It is a portable application

Based on web technologies

The application is portable : you can make it run on all the platform you need : Windows, Linux, Android, iOS, ...

Split your activities

Sessions

You can split your activities, so for example you can :

  • use a session for your professionnal activities
  • use a session for your sleep activity
  • use a session for your sport activity
  • ...

Easy to run

Simple as a click !

You just need to download static files (like html) and open it in your web-browser.

No need of a backend because data is stored in your browser.

Easy to save

Backup and portability is the crux of the battle !

You can save your activities in a file or use CouchDB replication protocol.

We can't see your data, so always keep in mind that you are responsible of your data !

Easy to understand

Google's Material Design

Build with a Material Design framework, the application looks like to be like any other.

Customizable

Custom fields by session

You can add custom fields for each session, so for example you can :

  • design an client field for your professionnal activities, where you store the name of the client you workend on this activity
  • design an quality field for your sleep activity, so you can rate the quality of your sleep
  • ...

Analytics

Charts

Filter your data with your custom fields and render them in sweet charts.

Otherwise, why collect all this data ?

Internationalization

Translated Interface

All the items of the User Interface can be translated.
